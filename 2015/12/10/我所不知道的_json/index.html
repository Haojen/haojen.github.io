你所不知道的 JSON
  1. JSON 里面的字符串必须使用双引号，而如果使用单引号的话会导致语法错误
  2. 不支持 undefined
  3. JSON 没有变量概念
  4. 没有末尾的分号，因为这不是 JS 语句，所以不需要分号
  5. 对象的属性必须加双引号
  6. JSON 对象有两个方法：stringify() 和parse() , 这两个方法分别用于把 JS 对象序列化为 JSON 字符串和把 JSON 字符串解析为原生 JS 值。
